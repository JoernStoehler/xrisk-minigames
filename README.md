# X-Risk Minigames

Small web apps that communicate AI existential risk concepts.

Each project lives in `projects/<name>/` as a self-contained Vite + React + TypeScript app.

## Getting Started

```bash
# Create a new project from template
cp -r projects/_template projects/your-project-name
cd projects/your-project-name
npm install
npm run dev
```
